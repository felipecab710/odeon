# Set preview uses timeline clips (interim)

Set Builder preview (`odeon-set-preview`) loads each lane as a route with clips at timeline positions — overlapping clips mix like a DAW arranger, not like four independent CDJ deck players. Booth preview uses true deck players (`createDjSession`). This interim model ships Set Builder before full deck parity; beat-sync and CDJ-feel work is blocked until set preview converges on deck players or a bus-based crossfader graph.

**Status:** accepted (interim). Supersede when Phase 1C deck-player parity lands.
