Odeon v0.1 — Install

1. Drag Odeon.app to Applications.
2. Eject this disk image.
3. Open Terminal and run:

   xattr -cr /Applications/Odeon.app

   (macOS blocks unsigned downloads as "damaged" until quarantine is cleared.)

4. Open Odeon from Applications.
5. Install ffmpeg for MP3/AAC: brew install ffmpeg

Full guide: https://github.com/felipecab710/odeon/blob/main/docs/INSTALL.md
